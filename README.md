# Telco-Customer-Churn-analysis

Customer Churn Analysis

Project Overview
This project analyzes customer churn data to identify patterns and key factors affecting customer retention. The goal is to understand why customers leave and how businesses can improve retention strategies.

Tools & Skills Used:
Microsoft Excel,
Data Cleaning,
Pivot Tables,
Data Visualization

Project Files
Customer_Churn_Analysis.xlsx – Contains raw data, pivot tables, and dashboard

dashboard.png – Visual representation of insights

Dashboard:

Key Analysis Performed:

Calculated overall churn rate

Analyzed churn based on contract type

Studied churn patterns across customer tenure

Key Insights:

1.Customers with month-to-month contracts have the highest churn rate.

2.Customers with shorter tenure (0–12 months) are more likely to churn.

3.Long-term customers are more likely to stay.

Conclusion
The analysis highlights important trends in customer behavior and provides insights that can help businesses improve customer retention and reduce churn.
